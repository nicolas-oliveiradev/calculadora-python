# Calculadora em Python

## Descrição
Calculadora em Python executada via terminal, com:
- Menu dinâmico usando dicionários
- Validação de entradas
- Funções matemáticas (soma, divisão, raiz, seno, etc.)
- Reutilização do último resultado

## Tecnologias
- Python 3
- Biblioteca math

## Como executar
```bash
python main.py
